# Decodelabs_Task1

## Project Overview

This repository contains my **Task 1: Data Cleaning and Feature Engineering** completed as part of the **DecodeLabs Data Science Internship**.

The project focuses on preparing raw data for machine learning by handling missing values, treating outliers, encoding categorical features, removing highly correlated variables, and creating new features. The final processed dataset is clean, consistent, and ready for model training.

## Data Processing Workflow

### 1. Data Loading

* Loads the Titanic dataset from an online source.
* Generates a sample dataset if the download fails.

### 2. Missing Value Handling

* Drops rows with very few missing values (<5%).
* Fills moderate missing values (5%–20%) using the median.
* Uses **KNN Imputer** for columns with higher missing percentages.
* Fills missing categorical values using the mode.

### 3. Outlier Treatment

* Detects outliers using the **Interquartile Range (IQR)** method.
* Caps extreme values using `numpy.clip()`.
* Excludes ID and target columns from outlier treatment.

### 4. Feature Encoding

* Applies **One-Hot Encoding** to low-cardinality categorical variables.
* Uses **Label Encoding (Factorization)** for high-cardinality categorical variables.

### 5. Correlation Analysis

* Identifies highly correlated numerical features.
* Removes features with a correlation greater than **0.80**, while preserving the target column.

### 6. Feature Engineering

* Creates interaction features.
* Generates ratio features.
* Computes Z-score normalized features.
* Applies logarithmic transformation to highly skewed numerical columns.

### 7. Final Validation

* Confirms that no missing values remain.
* Saves the cleaned dataset as **processed_data.csv**.
